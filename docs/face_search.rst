.. _face-search:

*******************************************
Идентификация лиц по базам данных
*******************************************

FindFace Security позволяет выполнять идентификацию (поиск) лиц по следующим базам данных:

* База данных обнаруженных на видео лиц (вкладка :guilabel:`События`).
* База данных досье (вкладка :guilabel:`Досье`). Содержит эталонные изображения лиц.
* Социальная сеть vk.com.

Поиск лиц выполняется на вкладке :guilabel:`Поиск`.
